Assijus - Assinador Digital

Assinador Digital da Justiça - é um site que unifica documentos que precisam ser assinados digitalmente em diferentes sistemas, simplificando, e agilizando todo o processo de coleta de assinaturas.

Instalação Certificado Digital A1

Certificados Digitais do tipo A1 são arquivos instalados em cada dispositivo a ser utilizado a partir de um arquivo que, geralmente, possui a extensão .pfx ou .p12. Veja abaixo como instalar, de acordo com seu dispositivo.

Android Mac OS

Android

Primeiro passo é transferir o arquivo .pfx ou .p12 para o dispositivo. Veja uma das formas abaixo:

  1. Conecte o celular ao computador
  2. No celular, clique em PERMITIR para possibilitar a cópia de arquivos entre computador e celular.
  3. Selecione a opção Transferir arquivos.
  4. No computador, acesse a pasta onde foram salvos os arquivos e COPIE para uma pasta dentro do celular, o Certificado criado.
  5. Desconecte o telefone do computador.

Depois o próximo passo é localizar, extrair e instalar o certificado no celula:

  1. Localize no celular o arquivo do Certificado .pfx ou .p12 copiado e toque para abrir.
  2. Digite a senha criada e clique em OK
  3. Defina um Nome do Certificado e clique em OK

Com isso seu Certificado Digital A1 estará pronto para uso.

  1. Para testar, instale e abra o aplicativo Assijus disponível na Play Store.
  2. Selecione o Certificado conforme nome definido e localize documentos disponíveis para assinatura.

Serão listados os documentos disponíveis para assinatura, conforme o Certificado Digital carregado.

Ainda tem dúvidas, acesse o Tutorial e saiba como usar.

Obs.: Para Tablets, realizar o mesmo procedimento acima, porem caso ocorra erro:

  1. Realizar o procedimento para o celular.
  2. Ativar o bluetooth do celular e do tablet.
  3. Clicar sobre o certificado que esta no celular.
  4. Escolha a opção compartilhar via bluetooth e escolha o tablet.
  5. Repetir os passos a partir do item 6.

Mac OS

Instalar o Certificado Digital no Mac OS é simples e rápido.
O arquivo com seu certificado (.pfx ou .p12) deve estar disponível no seu computador.
Caso ainda não esteja, envie o arquivo para seu computador.

  1. Acesse o local onde encontra-se o arquivo com o certificado.
  2. Dê um duplo clique no arquivo .pfx ou .p12 para iniciar a importação do certificado.
  3. Será aberto o gerenciador de chaves do Mac OS. Informe a senha que foi definida na exportação do seu Certificado Digital e clique em OK para instalar.

Após instalado, seu Certificado Digital estará disponível para utilização.

Para utilizar, acesse o menu Documentos e, quando solicitado, selecione o Certificado Digital A1.

Por padrão, todas as vezes que o sistema precisar acessar sua chave para autenticar-se ou produzir as assinaturas, será solicitada sua senha de Login do computador .

Informe a senha e clique em Permitir

Serão listados os documentos disponíveis para assinatura, conforme o Certificado Digital carregado.

Ainda tem dúvidas, acesse o Tutorial e saiba como usar.
