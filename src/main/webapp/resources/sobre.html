Por que Assijus?

O Assijus - Assinador Digital da Justiça - é um site que unifica documentos que precisam ser assinados digitalmente em diferentes sistemas, simplificando, e agilizando todo o processo de coleta de assinaturas.

No primeiro acesso ao Assijus, em cada equipamento, é necessária uma instalação prévia, conforme as orientações abaixo:

Passo a passo

  1. Instale o driver do token para acessar seu certificado.
  2. Instale o programa {{assijusexe}} em seu PC/Windows. Este programa depende do Microsoft .NET Framework 4.5.1, e talvez seja necessário instalá-lo também.
  3. Retorne para o site do Assijus e vá para o menu de DOCUMENTOS
  4. Escolha o certificado se houver mais de um e informe o PIN
  5. Marque ou desmarque documentos para indicar quais devem ser assinados
  6. Clique em "Assinar documentos marcados"
