Assijus - Assinador Digital

Assinador Digital da Justiça - é um site que unifica documentos que precisam ser assinados digitalmente em diferentes sistemas, simplificando, e agilizando todo o processo de coleta de assinaturas.

Instalação

No primeiro acesso ao Assijus, em cada equipamento, é necessária uma instalação prévia, conforme as orientações abaixo.

Para Certificados Digital A3, armazenados em Token ou Smartcards, caso o equipamento nunca tenha sido utilizado para assinar documentos, pode ser necessário instalar o driver do token e a cadeia de certificados conforme orientação da Autoridade Certificadora utilizada.

Para Certificados Digital A1, armazenados no próprio dispositivo, eles devem ser emitidos e instalados conforme Sistema Operacional e orientações da Autoridade Certificadora.

Clique aqui e saiba como importar e instalar o Certificado Digital A1 em seu dispositivo.

O Assijus Web é compatível apenas com o navegador Google Chrome, através da instalação de uma Chrome Extension. Para dispositivos móveis utilize a Versão mobile

Windows

No sistema operacional Windows é necessário que o computador tenha previamente instalado o Microsoft .NET Framework 4.5.1

Instale a aplicação assijus-chrome-extension-setup.msi para que o Assijus acesse seu Certificado Digital.

Após efetuar a instalação da aplicacação, encerre todas as janelas do Google Chrome e abra-o novamente. O Assijus estará pronto para uso

Caso a extensão não seja detectada, verifique se está instalada acessando a página da Chrome Web Store . Também verifique na Lista de Extensões (Menu/Mais ferramentas/Extensões) se o Assijus Signer está presente e ativo.

Mac OS

No sistema operacional Mac OS é necessário que o computador tenha previamente instalado o Java 8

Instale a aplicação assijus-chrome-extension-setup.pkg para que o Assijus acesse seu Certificado Digital.

Caso receba uma mensagem semelhante à: "assijus-chrome-extension-setup.pkg não pode ser aberto porque procede de um desenvolvedor não identificado", favor seguir as seguintes orientações da Apple disponível em: Clique aqui

Após efetuar a instalação da aplicacação, encerre todas as janelas do Google Chrome e abra-o novamente. O Assijus estará pronto para uso

Caso a extensão não seja detectada, verifique se está instalada acessando a página da Chrome Web Store . Também verifique na Lista de Extensões (Menu/Mais ferramentas/Extensões) se o Assijus Signer está presente e ativo.

Versão mobile

Faça o download do Assijus para assinar com Certificado A1 no celular e produza assinaturas digitais no padrão ICP-Brasil AD-RB, utilizando certificado digital A1 diretamente de seu celular e em qualquer lugar.

Clique aqui e saiba como importar o Certificado Digital A1 para seu celular.

Após instalado os pré-requisitos do Assijus e seu Certificado Digital, acesse seus Documentos para listar e assinar.
Ainda tem dúvidas, acesse o Tutorial e saiba como usar.
